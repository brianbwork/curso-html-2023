Etiquetas de linea

Etiquetas

"em" es la etiqueta que se pone dentro de una etiqueta "p" para dar emfacis al texto o palaba

Strong

la etiqueta "strong" da mas enfacis que la etiqueta "eM" y se utilza de la misma maneta ademas de que se pone en color negrita el texto o palabra

Small

la etiqueta "small"sirve para quitarle importancia al texto o palabra seleccionada incluso menos que el texto en general

Br

la etiqueta "br" sirve para hacer un salto de linea
separar el texto deseado

Wbr

la etiqueta "wbr" un ejemplo en donde se puede utilizar es en los url (palabras largas) y es para especificar en que parte se quiere que se separe o parta la palabra cuando se modifica el espacio en donde se encuentra

Time

la etiqueta "time" se utiliza para aclarar al servidor que el texto dentro de esta eqiqueta indica que es un formato de fecha y hora y se puede poner ambas al mismo tiempo o solo una

Etiquetas de ( estilo en concreto "i", "b", "u" )que sirven para dejar por debajo de la linea o por arriba

estiqueta "i" que se refiere a italic

etiqueta "i" se utlizapr poner iconos en librerias

etiqueta "b" bold

"b" que se refiere a bold que lo convierte el texto solo en negrita

etiqueta "u" underline

etiqueta "u" que se refiere a texto subrayado

etiqueta "sup" superindice

la etiqueta "sup" se utiliza para poner letras o numeros dentro de un texto por arriba de la linea normal de esta manera 22

Etiqueta "sub" subindice

la etiqueta "sub" subindice se utiliza para poner por debajo de la linea del texto tanto palabras como letras o numeros de esta manera H2O2

Atributos globales

Veremos varios atributos que harán de puente para la utilizar mas adelante tanto CSS como JS

Class: El atributo "class" se utilza para dar estilos a travez de css

Id: El atributo "Id" es un identificador unico con el que estando en js se podra seleccionar los elementos generar conecciones en el servidor o navegar a travez de anclas)

Style: El atributo "Style" es donde se seccionara la palabra o bloque referidos a los que se quiere modificar

Title: El atributo "Title" se utilza para dejar un mensaje o informacion un texto o plabra , la misma se abrirá en formato de texto proporcionando informacion a la palabra que marcamos con el mouse (tambien nombrado como tooltip)

Data: El atributo "Data" se utilza para guardar algun "valor" a en etiquetas html esto es un ejemplo de como utilizar
