Etiquetas de linea

Etiquetas

"es" es la etiqueta que se pone dentro de una etiqueta "p" para dar emfacis al texto o palabas

Strong

la etiqueta "strong" da mas enfacis que la etiqueta "es" y se utilza de la misma maneta ademas de que se pone en color negrita el texto o palabra

Small

la etiqueta "small"sirve para quitarle importancia al texto o palabra seleccionada incluso menos que el texto en general

Br

la etiqueta "br" sirve para hacer un salto de linea
separar el texto deseado

Wbr

la etiqueta "wbr" un ejemplo en donde se puede utilizar es en los url (palabras largas) y es para especificar en que parte se quiere que se separe o parta la palabra cuando se modifica el espacio en donde se encuentra

Time

la etiqueta "time" se utiliza para aclarar al servidor que el texto dentro de esta eqiqueta indica que es un formato de fecha y hora y se puede poner ambas al mismo tiempo o solo una
